# Task 1 - To-Do List Application

## CodeSoft Python Programming Internship

### Project Overview

This project is a command-line To-Do List application developed using Python. It helps users organize and manage their daily tasks efficiently.

### Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Exit the application

### Technologies Used

* Python
* Functions
* Lists
* Dictionaries
* Loops
* Conditional Statements
* Exception Handling

### How to Run

1. Clone the repository.
2. Open the project folder.
3. Run the following command:

python todo_list.py

4. Follow the menu options to manage tasks.

### Sample Output

1. Add Task
2. View Tasks
3. Mark Task as Completed
4. Delete Task
5. Exit
